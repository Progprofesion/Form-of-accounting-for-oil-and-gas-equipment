# React App
Выполненно в минималистическом стиле и с минимальным количеством зависимостей.

## Technology stack used

Axios,  
React-hook-form,  
React-redux,  
@reduxjs/toolkit,  
react-router-dom,    
Scss,  

### `Launch`

npm i , npm start
